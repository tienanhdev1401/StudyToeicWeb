# Hướng dẫn tạo file Excel cho Vocabulary Topic

## Cấu trúc file Excel

File Excel để import từ vựng cần có cấu trúc sau:

| content | meaning | synonym | transcribe | urlAudio | urlImage |
|---------|---------|---------|------------|----------|----------|
| hello   | xin chào | hi, hey | həˈloʊ     | URL      | URL      |
| world   | thế giới | earth   | wɜrld      | URL      | URL      |

## Giải thích các cột

1. **content**: Từ vựng (bắt buộc)
2. **meaning**: Ý nghĩa của từ (bắt buộc)
3. **synonym**: Từ đồng nghĩa, có thể là một từ hoặc nhiều từ cách nhau bởi dấu phẩy (tùy chọn)
4. **transcribe**: Phiên âm của từ (tùy chọn)
5. **urlAudio**: Đường dẫn đến file âm thanh (tùy chọn)
6. **urlImage**: Ảnh của từ 

## Lưu ý quan trọng

- Tên cột phải chính xác như trên (phân biệt chữ hoa, chữ thường)
- File Excel phải có ít nhất 1 hàng dữ liệu
- Các cột content và meaning là bắt buộc, các cột khác có thể để trống
- Các từ đồng nghĩa nên cách nhau bởi dấu phẩy
- Hệ thống cũng hỗ trợ tên cột tiếng Việt: "Từ", "Nghĩa", "Từ đồng nghĩa", "Phiên âm", "Audio", "Hình"
